# Hiring Process Analytics

This project involves analyzing the company's hiring process data to uncover meaningful insights. The hiring process is a critical function of any organization, and understanding patterns such as rejection rates, interview trends, job types, and vacancies can provide valuable information for improving the hiring process.

## Objectives

1. Handling Missing Data:
   - Checked for missing values in the dataset.
   - Applied appropriate strategies to handle them .

2. Clubbing Columns:
   - Simplified analysis by combining columns with multiple categories where applicable.

3. Outlier Detection and Handling:
   - Identified outliers that could skew the analysis.
   - Applied suitable strategies to handle outliers, such as removing or replacing them.

4. Data Summary:
   - Summarized the data after cleaning and preparation.
   - Calculated statistical measures such as averages, medians, and other relevant metrics.
   - Created visualizations for better understanding of the data trends.

## Key Insights

- Trends in rejection rates and interview processes.
- Analysis of job types and vacancy patterns.
- Insights into potential improvements in the hiring process.

## Tools and Techniques Used

- Statistics: For data cleaning, analysis, and summarization.
- Excel: For data handling, visualization, and summarization.

## Outcome

The findings from this project provide actionable insights for the hiring department, helping the company to refine its hiring strategies and make more informed decisions in the future.
